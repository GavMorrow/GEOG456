# GEOG456
Data Analysis for a class on Internal Migration within the United States

Note: Data used is all from IPUMS with the exception of some IRS data used for county-county comparison in the Assignment 3.
Data files were too large to be used but usa_00001 should correspond with IPUMS ACS 2023 data, and usa_00007 is specific to the final project choosing 2006, 2016, 2021, and 2024 with variables related to degree and job type in an effort to define tech jobs as to focus on tech migration.
